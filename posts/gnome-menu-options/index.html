Gnome Menu Options

2021-05-09

Add/Replace Gnome Menu Options

Want to replace the Open in Terminal with Open in Alacritty (or any other terminal)?

What you need:

  1. FileManager Actions through:

    sudo pacman -S filemanager-actions

Steps

  1. Install the FileManager Actions.

  2. Launch the application and define a new action.

  3. Select display item for both selection and location in context menu.

  4. Action Tab settings:

    1. Context label: the text of the action which will appear in the context menu.

  5. Command Tab settings:

    1. Path: /usr/bin/alacritty (replace alacritty with your terminal of choice.)

    2. Parameters: --working-directory=%d/%b

  6. Save the newly created action!

Congratulations!
