Switching from Deepin DE to KDE

This tutorial is written for Arch Linux users and should be followed with caution!

Getting Started

This is a guide for people using Deepin DE and doesn't know how to switch to KDE without breaking the system or having to reinstall Arch Linux.

Installing Packagesnew

Install the packages required by your new Desktop Environemnt (DE) of choice you want to replace Deepin with, which in this case is KDE Plasma:

  • sudo pacman -S plasma-meta

  • Plasma-meta: Install all the packages without being able to exclude some packages that will be installed.

  • Plasma: Install all the desired packages by selecting which packages you want to install, write the number of the packages and later click enter for them to install.

  • Plasma-desktop: the minimal version of KDE that will install all the nessecary files to make the DE run.

Install KDE applications

  • sudo pacman -S kde-applications-meta

  • KDE-applications: Install the desired packages.

  • KDE-applications-meta: Install all the packages without being able to control which packages won't be installed.

Uninstall Packages of Current DE

  • sudo pacman -Runs deepin deepin-extra

CTRL+ALT+F2 => Login as ROOT user.

Run the following commands:

  • systemctl disable ligthdm.service

  • systemctl enable sddm.service

  • systemctl start sddm.service

Congratulations! You have just successfully installed KDE display manager and removed Deepin DE!
